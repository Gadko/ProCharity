Приватный профиль будет скрыт для неавторизованных пользователей в списке волонтеров и в поиске по сайту

Мой профиль виден только пользователям ProCharity

Если ты не хочешь больше быть волонтером ProCharity, можешь заблокировать свой профиль. Это значит:

  • ты не сможешь откликаться на задания;
  • твой профиль будет скрыт для всех пользователей
  • ты не будешь получать от нас никаких писем.
